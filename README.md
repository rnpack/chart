# @rnpack/charts

Various charts with animation for React Native

## Installation

```sh
npm install @rnpack/charts
```

## Dependencies

```sh
npm i react-native-svg
```

## Usage

```js
import { DonutChart } from '@rnpack/charts';

// ...

<DonutChart radius={45} />
```
